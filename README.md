# Virtual Pet Simulator – Java Console Game

A fun, interactive console-based virtual pet game written in Java! Adopt up to 3 pets (dogs, cats, or fish), take care of them by feeding, playing, resting, and enjoying their unique behaviors. Neglect them and they’ll get sick or die — time passes, stats decay, and random events keep things exciting!

Built to showcase solid Object-Oriented Programming concepts in Java.

## Features
- Multiple users, each can own up to 3 pets
- 3 pet types: Dog, Cat, Fish (each with unique behaviors)
- Real-time stat decay (hunger ↑, energy ↓, health ↓ each turn)
- Random events (positive surprises or minor setbacks)
- Colored console output for better readability
- Custom pet naming
- Sickness (health < 20) and death mechanics
- Adopt from a shared pool of available pets
- Simulate damage to test pet survival

## Technologies
- Java (JDK 8+)
- Scanner for input
- ArrayList for dynamic collections
- OOP: abstract classes, inheritance, polymorphism, encapsulation
- ANSI escape codes for colored terminal text
